This project visualizes the working of an **LRU (Least Recently Used) Cache**
using HTML, CSS, and JavaScript.

## Features
- Live cache operations
- PUT and GET functionality
- Visual representation of cache order
- Cache HIT & MISS indication
- Beginner-friendly implementation

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
- Most recently used items move to the right
- Least recently used items are removed when capacity is full
